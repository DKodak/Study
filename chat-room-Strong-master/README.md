# 基于spring Boot + spring WebSocket的在线聊天室

### 1. 目的

一方面加强自己的代码能力，同时巩固所学，另一方面想将之前的图片服务器项目利用起来。

### 2. 技术

* SpringBoot(Sping MVC + spring WebSocket + Spring Core + Spring JDBC)

* MySQL数据库
* html 5 + CSS + jQuery

### 3. 功能

* 私聊
* 群聊
* 发送图片
